## ğŸ“– Overview

This project is a client-side web application built with React, Vite, and Tailwind CSS, designed to facilitate online donations for various causes. It provides a simple and intuitive interface for users to browse campaigns, learn about organizations, and contribute financially.  The application is deployed on Firebase Hosting.

## âœ¨ Features

- **Campaign Browsing:** View a list of active fundraising campaigns with details and images.
- **Campaign Details:** See in-depth information about each campaign, including its goals, progress, and the organization behind it.
- **Secure Donations:**  Make donations through a secure payment gateway (integration details not provided in this repository).
- **Responsive Design:**  Accessible and user-friendly across different devices (desktops, tablets, and mobile phones).

## ğŸš€ Quick Start

### Prerequisites

- Node.js (version >=16.0.0)
- npm (or yarn)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Eagl3Eyes/easy-fund-client-side.git
   cd easy-fund-client-side
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Visit `http://localhost:5173`
